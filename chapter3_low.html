Question 1: Find \(\delta y\) and \(dy\)

Remember the difference:
  • \(\delta y\) (delta y) is the **actual change** in \(y\). We find it by \(\delta y = f(x + \delta x) - f(x)\).
  • \(dy\) is the **approximate change** in \(y\). We find it using the derivative: \(dy = f'(x) dx\).

(i) \(y = x^2 - 1\), when \(x\) changes from 3 to 3.02

1 Identify the given values.
Our function is \(f(x) = x^2 - 1\).
The starting \(x\) is 3. The ending \(x\) is 3.02.
So, \(x = 3\).
The change in \(x\), which we call \(\delta x\) (or \(dx\)), is \(3.02 - 3 = 0.02\).
2 Calculate \(\delta y\) (the actual change).
\(\delta y = f(x + \delta x) - f(x) = f(3.02) - f(3)\)
First, find \(f(3.02)\): \(f(3.02) = (3.02)^2 - 1 = 9.1204 - 1 = 8.1204\).
Next, find \(f(3)\): \(f(3) = (3)^2 - 1 = 9 - 1 = 8\).
Now, subtract: \(\delta y = 8.1204 - 8 = 0.1204\).
3 Calculate \(dy\) (the approximate change).
\(dy = f'(x) dx\)
First, find the derivative of \(f(x)\): \(f'(x) = 2x\).
Now, plug in our starting \(x = 3\): \(f'(3) = 2(3) = 6\).
We already know \(dx = \delta x = 0.02\).
Calculate \(dy\): \(dy = (6)(0.02) = 0.12\).
Final Answer: \(\delta y = 0.1204\) and \(dy = 0.12\). (Notice how close they are!)

(ii) \(y = x^2 + 2x\), when \(x\) changes from 2 to 1.8

1 Identify the given values.
Our function is \(f(x) = x^2 + 2x\).
The starting \(x\) is 2.
The change in \(x\), \(\delta x\), is \(1.8 - 2 = -0.2\). (It's negative because \(x\) decreased).
2 Calculate \(\delta y\).
\(\delta y = f(1.8) - f(2)\)
\(f(1.8) = (1.8)^2 + 2(1.8) = 3.24 + 3.6 = 6.84\).
\(f(2) = (2)^2 + 2(2) = 4 + 4 = 8\).
\(\delta y = 6.84 - 8 = -1.16\).
3 Calculate \(dy\).
\(dy = f'(x) dx\)
Derivative: \(f'(x) = 2x + 2\).
Plug in \(x = 2\): \(f'(2) = 2(2) + 2 = 6\).
We know \(dx = -0.2\).
Calculate \(dy\): \(dy = (6)(-0.2) = -1.2\).
Final Answer: \(\delta y = -1.16\) and \(dy = -1.2\).

(iii) \(y = \sqrt{x}\), when \(x\) changes from 4 to 4.41

1 Identify the given values.
\(f(x) = \sqrt{x}\). \(x = 4\). \(\delta x = 4.41 - 4 = 0.41\).
2 Calculate \(\delta y\).
\(\delta y = f(4.41) - f(4)\)
\(f(4.41) = \sqrt{4.41} = 2.1\).
\(f(4) = \sqrt{4} = 2\).
\(\delta y = 2.1 - 2 = 0.1\).
3 Calculate \(dy\).
\(dy = f'(x) dx\)
Derivative: \(f'(x) = \frac{1}{2\sqrt{x}}\).
Plug in \(x = 4\): \(f'(4) = \frac{1}{2\sqrt{4}} = \frac{1}{2(2)} = \frac{1}{4} = 0.25\).
We know \(dx = 0.41\).
Calculate \(dy\): \(dy = (0.25)(0.41) = 0.1025\).
Final Answer: \(\delta y = 0.1\) and \(dy = 0.1025\).

Question 2: Using differentials find \(\frac{dy}{dx}\) and \(\frac{dx}{dy}\)

Key Idea:
  1. Take the differential, \(d(...)\), of both sides of the equation.
  2. Remember the rules: \(d(x^n) = nx^{n-1}dx\) and \(d(y^n) = ny^{n-1}dy\).
  3. Use the product rule: \(d(uv) = u \cdot dv + v \cdot du\).
  4. Algebraically separate all \(dx\) terms from all \(dy\) terms.
  5. Solve for \(\frac{dy}{dx}\). \(\frac{dx}{dy}\) will just be the reciprocal (flipped fraction).

(i) \(xy + x = 4\)

1 Take the differential of both sides.
\(d(xy + x) = d(4)\)
2 Apply differential rules. Use the product rule for \(xy\).
\(d(xy) + d(x) = 0\)
\((x \cdot dy + y \cdot dx) + 1 \cdot dx = 0\)
3 Group the \(dx\) and \(dy\) terms.
\(x \cdot dy + (y + 1) \cdot dx = 0\)
4 Solve for \(\frac{dy}{dx}\).
\(x \cdot dy = -(y + 1) \cdot dx\)
\(\frac{dy}{dx} = -\frac{y+1}{x}\)
5 Find \(\frac{dx}{dy}\) by flipping the fraction.
\(\frac{dx}{dy} = -\frac{x}{y+1}\)

Question 3: Use differentials to approximate

The Approximation Formula: We use the formula \(f(x + \delta x) \approx f(x) + dy\).
Since \(dy = f'(x) dx\), the full formula is:
\(f(x + \delta x) \approx f(x) + f'(x) dx\)

(i) \(\sqrt{17}\)

1 Choose \(f(x)\), \(x\), and \(dx\).
The function is \(f(x) = \sqrt{x}\).
We need a "nice" number near 17 that we can square root easily. Let's choose \(x = 16\).
The change from 16 to 17 is \(dx = 1\).
2 Find \(f(x)\) and \(f'(x)\).
\(f(x) = \sqrt{x} \implies f(16) = \sqrt{16} = 4\).
\(f'(x) = \frac{1}{2\sqrt{x}} \implies f'(16) = \frac{1}{2\sqrt{16}} = \frac{1}{2(4)} = \frac{1}{8} = 0.125\).
3 Use the approximation formula.
\(\sqrt{17} \approx f(16) + f'(16) dx\)
\(\sqrt{17} \approx 4 + (0.125)(1)\)
\(\sqrt{17} \approx 4.125\)

Question 4: Approximate increase in the volume of a cube

Find the approximate increase in the volume of a cube if the length of its each edge changes from 5 to 5.02.

1 Define the function and variables.
We are talking about Volume, so the function is \(V(s) = s^3\), where \(s\) is the side length.
The starting side length is \(s = 5\).
The change in side length is \(ds = 5.02 - 5 = 0.02\).
We need to find the "approximate increase," which means we need to find \(dV\).
2 Find the differential \(dV\).
\(V = s^3\)
Take the derivative with respect to \(s\): \(\frac{dV}{ds} = 3s^2\)
Now, solve for \(dV\): \(dV = 3s^2 \cdot ds\)
3 Plug in the values for \(s\) and \(ds\).
\(dV = 3(5)^2 \cdot (0.02)\)
\(dV = 3(25) \cdot (0.02)\)
\(dV = 75 \cdot (0.02) = 1.5\)
Final Answer: The approximate increase in volume is 1.5 cubic units.

Question 5: Evaluate Indefinite Integrals

This next set of problems involves **Indefinite Integration** (also called finding the 'anti-derivative'). This is the reverse process of differentiation.

Key Concepts to Remember:
  • Power Rule: \(\int x^n dx = \frac{x^{n+1}}{n+1} + C\) (as long as \(n \neq -1\))
  • Algebraic Simplification: Always try to expand, split fractions, or rewrite the problem *before* integrating.
  • U-Substitution: Look for a function (like \(\ln x\)) whose derivative (\(\frac{1}{x} dx\)) is also in the integral.

(i) \(\int(\sqrt{x} + 1)^2 dx\), (x > 0)

1 Simplify the expression.
We can't integrate this directly. Let's expand the square first using \((a+b)^2 = a^2 + 2ab + b^2\).
\((\sqrt{x} + 1)^2 = (\sqrt{x})^2 + 2(\sqrt{x})(1) + (1)^2\)
\( = x + 2\sqrt{x} + 1\)
2 Rewrite the integral.
Now we can integrate this simpler polynomial.
\(\int (x + 2\sqrt{x} + 1) dx\)
3 Rewrite using fractional exponents.
To use the power rule, we write \(\sqrt{x}\) as \(x^{1/2}\).
\(\int (x^1 + 2x^{1/2} + 1) dx\)
4 Integrate each term using the Power Rule.
\(\int x^n dx = \frac{x^{n+1}}{n+1}\)
\(\int x^1 dx = \frac{x^{1+1}}{1+1} = \frac{x^2}{2}\)
\(\int 2x^{1/2} dx = 2 \cdot \frac{x^{1/2 + 1}}{1/2 + 1} = 2 \cdot \frac{x^{3/2}}{3/2}\)
\(\int 1 dx = x\)
5 Simplify and add the constant of integration, \(C\).
Simplifying the middle term: \(2 \cdot \frac{x^{3/2}}{3/2} = 2 \cdot \frac{2}{3}x^{3/2} = \frac{4}{3}x^{3/2}\).
Final Answer: \(\frac{x^2}{2} + \frac{4}{3}x^{3/2} + x + C\)

(ii) \(\int (\ln x) \cdot \frac{1}{x} dx\), (x > 0)

Hint: Use U-Substitution!

This problem is perfect for u-substitution. We look for a function (which we'll call \(u\)) whose derivative (\(du\)) is also present in the integral.

1 Identify \(u\) and \(du\).
Notice that the derivative of \(\ln(x)\) is \(\frac{1}{x}\). Both of these parts are in our integral!
So, let's set:
\(u = \ln(x)\)
Now, we find the differential \(du\) by taking the derivative of \(u\):
\(du = \frac{1}{x} dx\)
2 Substitute \(u\) and \(du\) into the integral.
Rewrite the original integral by replacing \(\ln(x)\) with \(u\) and \(\frac{1}{x} dx\) with \(du\).
\(\int \underbrace{(\ln x)}_{u} \cdot \underbrace{(\frac{1}{x} dx)}_{du} = \int u \cdot du\)
3 Integrate the new, simpler integral.
This is now a simple power rule problem.
\(\int u^1 du = \frac{u^{1+1}}{1+1} + C = \frac{u^2}{2} + C\)
4 Substitute back to \(x\).
Our final answer must be in terms of \(x\), not \(u\). Replace \(u\) with its original definition: \(\ln(x)\).
Final Answer: \(\frac{(\ln x)^2}{2} + C\)

Calculus Concepts Review

A breakdown of the key techniques from your problem sheet, with built-in breaks.

Concept 1: Integration by Parts

This technique is used to integrate products of functions. It comes from the product rule for differentiation: \(d(uv) = u\;dv + v\;du\).

\(\int u\;dv = uv - \int v\;du\)

Steps to Solve:

  1. Identify u and dv: Look at your integral (e.g., \(\int x \sin(x) dx\)). You must split it into two parts. A good strategy is the LIATE rule. Choose u based on whichever function type appears first in this list:
    • Logarithmic (e.g., \(\ln(x)\))
    • Inverse Trig (e.g., \(\tan^{-1}(x)\))
    • Algebraic (e.g., \(x^2\), \(x^3\))
    • Trigonometric (e.g., \(\sin(x)\))
    • Exponential (e.g., \(e^x\))
  2. Calculate du and v: Find du by differentiating u. Find v by integrating dv.
  3. Apply the Formula: Plug your four pieces (u, v, du, dv) into the formula: \(uv - \int v\;du\).
  4. Solve the New Integral: Solve the new, hopefully simpler, integral \(\int v\;du\).

Concept 2: Trigonometric Substitution

This technique is used for integrals containing specific square root forms.

Steps to Solve:

  1. Identify the Form:
    • If you see \(\sqrt{a^2 - x^2}\): Use substitution \(x = a \sin(\theta)\).
    • If you see \(\sqrt{x^2 - a^2}\): Use substitution \(x = a \sec(\theta)\).
    • If you see \(\sqrt{a^2 + x^2}\): Use substitution \(x = a \tan(\theta)\).
  2. Substitute: Replace both \(x\) and \(dx\) in the integral. (e.g., if \(x = a \sin(\theta)\), then \(dx = a \cos(\theta) d\theta\)).
  3. Simplify: Use trigonometric identities (like \(1 - \sin^2(\theta) = \cos^2(\theta)\)) to eliminate the square root.
  4. Integrate: Solve the new integral with respect to \(\theta\).
  5. Convert Back to x: Use your original substitution (e.g., \(\sin(\theta) = x/a\)) to draw a right-angled triangle. Use SOH CAH TOA to find expressions for the trig parts of your answer (like \(\cos(\theta)\)) in terms of \(x\).

Concept 3: Trigonometric Integrals

This involves integrals with powers of trigonometric functions, like \(\int \sec^4(x) \tan^2(x) dx\).

Steps to Solve (Powers of sec and tan):

  1. Case 1: Secant power is EVEN:
    • Save one \(\sec^2(x)\) factor (this will be your du).
    • Use the identity \(\sec^2(x) = 1 + \tan^2(x)\) to convert all other sec factors into tan.
    • Use u-substitution with \(u = \tan(x)\) and \(du = \sec^2(x) dx\).
  2. Case 2: Tangent power is ODD:
    • Save one \(\sec(x) \tan(x)\) factor (this will be your du).
    • Use the identity \(\tan^2(x) = \sec^2(x) - 1\) to convert all other tan factors into sec.
    • Use u-substitution with \(u = \sec(x)\) and \(du = \sec(x) \tan(x) dx\).

Concept 4: u-Substitution

This is the most fundamental substitution method, often used to simplify other problems.

Steps to Solve:

  1. Identify u: Look for an "inside function" whose derivative is also present in the integral (perhaps off by a constant).
    • In \(\int x^2 e^{x^3} dx\), the "inside function" is \(u = x^3\).
  2. Calculate du: Differentiate your u.
    • If \(u = x^3\), then \(du = 3x^2 dx\).
  3. Substitute: Rearrange your du equation to match the integral.
    • We have \(x^2 dx\) in the integral. From step 2, we know \(x^2 dx = du / 3\).
    • The integral becomes \(\int e^u (du / 3)\).
  4. Integrate: Solve the new, simpler integral.
    • \(\frac{1}{3} \int e^u du = \frac{1}{3}e^u + C\)
  5. Convert Back to x: Replace u with its original expression in x.
    • \(\frac{1}{3}e^{x^3} + C\)

Integration: Partial Fractions

This is the main technique for integrating rational functions (one polynomial divided by another).

Concept 1: Improper Fractions (Long Division)

This is your **first step** if the numerator's degree is greater than or equal to the denominator's. (e.g., Q3, Q8).

Steps to Solve:

  1. Perform Polynomial Long Division. This will rewrite your fraction as:
    \[\text{Quotient} + \frac{\text{Remainder}}{\text{Divisor}}\]
  2. Integrate the Quotient. This is usually just a simple polynomial (e.g., \(\int (x+2) dx\)).
  3. Break Down the Remainder. The new fraction \(\frac{\text{Remainder}}{\text{Divisor}}\) is now *proper*, and you can use one of the other partial fraction methods on it.

Concept 2: Distinct Linear Factors

This is the most common case, where the denominator factors into unique, simple terms. (e.g., Q1, 2, 4, 5, 6, 7, 9, 10, 11).

Steps to Solve:

  1. Factor the denominator. For example, \(x^2 - x - 6 = (x-3)(x+2)\).
  2. Set up the fractions. Each factor gets its own fraction with a constant (A, B, C...) on top.
    \[\frac{3x+1}{(x-3)(x+2)} = \frac{A}{x-3} + \frac{B}{x+2}\]
  3. Solve for A and B. You can use:
    • Cover-Up Method: To find A, cover \((x-3)\) and plug \(x=3\) into the rest: \(A = \frac{3(3)+1}{(3+2)} = \frac{10}{5} = 2\).
    • Common Denominator: Multiply everything out and equate coefficients.
  4. Integrate. Your new integral is easy!
    \[\int \left( \frac{2}{x-3} + \frac{1}{x+2} \right) dx = 2\ln|x-3| + \ln|x+2| + C\]

Concept 3: Repeated Linear Factors

This is when a factor in the denominator is raised to a power. (e.g., Q13, 14, 15, 16, 17).

Steps to Solve:

  1. Factor the denominator. e.g., \((x+1)(x-2)^2\).
  2. Set up the fractions. The *repeated* factor gets one fraction for *each* power, from 1 up to its total power.
    \[\frac{...}{(x+1)(x-2)^2} = \frac{A}{x+1} + \frac{B}{x-2} + \frac{C}{(x-2)^2}\]
  3. Solve for A, B, and C. The cover-up method only works for A and C. You'll need to use the common denominator method or plug in "easy" values of x (like x=0) to find B.
  4. Integrate.
    \[\int \frac{A}{x+1} dx \rightarrow A\ln|x+1|\]
    \[\int \frac{B}{x-2} dx \rightarrow B\ln|x-2|\]
    \[\int \frac{C}{(x-2)^2} dx \rightarrow \int C(x-2)^{-2} dx = -C(x-2)^{-1} + C\]

Concept 4: Irreducible Quadratic Factors

This is when the denominator has a factor like \((x^2+1)\) that can't be factored further. (e.g., Q12, 18, 19, 20).

Steps to Solve:

  1. Factor the denominator. e.g., \((x+3)(x^2+1)\).
  2. Set up the fractions. The quadratic factor gets a *linear* numerator (\(Bx+C\)).
    \[\frac{9x-7}{(x+3)(x^2+1)} = \frac{A}{x+3} + \frac{Bx+C}{x^2+1}\]
  3. Solve for A, B, and C. Use the common denominator method. (Cover-up works for A).
  4. Integrate. The quadratic part will often split into two separate integrals:
    • One part you solve with u-substitution (e.g., \(\int \frac{Bx}{x^2+1} dx\)).
    • The other part is a standard \(\arctan\) integral (e.g., \(\int \frac{C}{x^2+1} dx\)).

Solving Definite Integrals

These problems introduce the **Fundamental Theorem of Calculus**. The main difference is the last step: evaluating the result at the upper and lower bounds.

Concept 1: The Fundamental Theorem of Calculus (FTC)

This is the core concept for *all* definite integrals. It connects integration and evaluation. (Applies to all problems, especially simple ones like Q1, 2, 8, 10, 11, 14).

\(\int_{a}^{b} f(x) dx = F(b) - F(a)\)

Steps to Solve:

  1. Find the Antiderivative: First, solve the indefinite integral \(\int f(x) dx\) just as you always would. Let's call the result \(F(x)\). You can ignore the \(+ C\) for definite integrals.
  2. Evaluate at Upper Bound: Plug the top number (\(b\)) into your result to get \(F(b)\).
  3. Evaluate at Lower Bound: Plug the bottom number (\(a\)) into your result to get \(F(a)\).
  4. Subtract: The final answer is \(F(b) - F(a)\).

Concept 2: U-Substitution (Definite)

When you use u-substitution in a definite integral, you have two ways to handle the bounds. (Applies to Q3, 4, 5, 6, 7, 9, 12).

Steps to Solve:

  1. Identify \(u\) and \(du\). (e.g., For Q7, \(u = x^2 + 2\), so \(du = 2x dx\))
  2. Choose Your Method:
    • Method A (Change the Bounds): This is often faster.
      1. Take your bounds (e.g., \(x=1\) and \(x=2\)) and plug them into your \(u\) equation to get *new* bounds in terms of \(u\).
      2. Example: If \(u = x^2+2\), then \(x=1 \rightarrow u = (1)^2+2 = 3\). And \(x=2 \rightarrow u = (2)^2+2 = 6\).
      3. Your new integral is \(\int_{3}^{6} \frac{1}{u} (\frac{1}{2}du)\). You *never* go back to \(x\).
    • Method B (Substitute Back):
      1. Solve the indefinite integral \(\int \frac{1}{u} (\frac{1}{2}du)\) to get \(\frac{1}{2}\ln|u|\).
      2. Substitute \(x\) back in: \(\frac{1}{2}\ln|x^2+2|\).
      3. Now, use your *original* \(x\)-bounds: \(\left[ \frac{1}{2}\ln|x^2+2| \right]_{1}^{2}\).

Concept 3: Advanced Integrals (Parts & Trig)

For complex problems, you use the same advanced techniques as before, and just apply the FTC at the very end. (Applies to Q13, 15, 16, 17, 18).

Steps to Solve:

  1. Integration by Parts (Q13):
    • Set \(u = \ln x\) and \(dv = dx\).
    • Find \(du = \frac{1}{x}dx\) and \(v = x\).
    • Apply the formula: \(\int \ln x dx = uv - \int v du = x\ln x - \int x(\frac{1}{x})dx = x\ln x - x\).
    • Apply FTC: \([x\ln x - x]_1^2 = (2\ln 2 - 2) - (1\ln 1 - 1) = 2\ln 2 - 1\).
  2. Trig Integrals (Q16, 18):
    • Use power-reduction formulas (e.g., \(\cos^2 t = \frac{1+\cos(2t)}{2}\)) to break the integral down.
    • Integrate the resulting simpler terms.
    • Apply FTC: \([F(\theta)]_a^b = F(b) - F(a)\). Be careful with radians!

Finding Area with Definite Integrals

This is the classic application of the Fundamental Theorem of Calculus. We are finding the exact area trapped between a function and the x-axis.

Concept 1: Area When \(f(x)\) is Above the x-axis

This is the most straightforward case. If the function's graph is entirely above the x-axis between your two bounds (\(a\) and \(b\)), the area is simply the value of the definite integral. (e.g., Q1, 2, 3, 4, 7, 10, 11, 12, 13).

\[\text{Area} = \int_{a}^{b} f(x) dx\]

Steps to Solve (Easy Words):

  1. 1. Set up your integral: Write down the integral \(\int\) with your "bottom" bound (\(a\)) at the bottom and your "top" bound (\(b\)) at the top. Put your function \(f(x)\) and \(dx\) inside.
  2. 2. Find the Antiderivative: Solve the integral just like you would an indefinite one (but you can ignore the \(+ C\)). Let's call your answer \(F(x)\).
  3. 3. Plug in the Bounds: Calculate \(F(b)\) (plugging in the top number) and \(F(a)\) (plugging in the bottom number).
  4. 4. Subtract: The final answer is \(F(b) - F(a)\).

Example Visualization (for Q1: \(\int_{1}^{2} (x^2 + 1) dx\)):

Imagine drawing the "U" shape of the parabola \(y = x^2 + 1\). Now, draw a vertical line up from \(x=1\) and another vertical line up from \(x=2\). The area you are solving for is the shape "trapped" between the curve, the x-axis, and these two lines.

Concept 2: Area When \(f(x)\) Crosses the x-axis

This is the trickier case. The integral will give a *negative* value for any area *below* the x-axis. Since "area" must be positive, you must find where the curve crosses, split the integral, and add the absolute values. (e.g., Q5, 6, 8, 9).

Steps to Solve (Easy Words):

  1. 1. Find the x-intercepts: Set your function \(f(x) = 0\) and solve for \(x\). These are your "split points."
  2. 2. Check your bounds:
    • If the problem gives you bounds (like Q8), see if any intercepts fall *between* them.
    • If no bounds are given (like Q5, 6, 9), your intercepts *are* the bounds.
  3. 3. Split the integral: If an intercept \(c\) is between your bounds \(a\) and \(b\), you must solve two separate integrals: \(\int_{a}^{c} f(x) dx\) and \(\int_{c}^{b} f(x) dx\).
  4. 4. Find the Absolute Value: Solve each integral. One of them will likely be negative. Take its absolute value (make it positive).
  5. 5. Add them together: Total Area = |Area 1| + |Area 2|.

Example Visualization (for Q5: \(y = 4x - x^2\)):

The intercepts are at \(x=0\) and \(x=4\). The graph is an "upside-down" parabola. The entire area is "trapped" between \(x=0\) and \(x=4\), and it's all above the x-axis, so you just solve \(\int_{0}^{4} (4x - x^2) dx\).

Differential Equations

These are equations that involve a function and its derivatives. We'll cover how to check, solve, and apply them.

Concept 1: Verifying Solutions & Solving Separable DEs

This covers most of your problems (Q1-18, 21). There are two main tasks: checking a given solution, or finding the solution from scratch.

Part A: Verifying a Solution (Q1)

This is just a check. You don't need to solve anything.

  1. 1. Find the Derivative: Take the given solution (e.g., \(y = cx - 1\)) and find its derivative (\(\frac{dy}{dx}\)). Here, \(\frac{dy}{dx} = c\).
  2. 2. Substitute: Plug both \(y\) and \(\frac{dy}{dx}\) into the *original* differential equation.
  3. 3. Check: The equation is \(x \frac{dy}{dx} = 1 + y\).
    Plugging in gives: \(x(c) = 1 + (cx - 1)\)
    Simplify: \(cx = 1 + cx - 1 \implies cx = cx\).
  4. 4. Conclude: Since it's true, the equation is a valid solution.

Part B: Solving Separable Equations (Most of Q2-18, 21)

This is the main method you'll use. The goal is to get all the \(y\)'s on one side and all the \(x\)'s on the other.

  1. 1. Separate: Get all \(y\) terms and \(dy\) on one side. Get all \(x\) terms and \(dx\) on the other.
    Example (Q3): \(y dx + x dy = 0 \implies x dy = -y dx\)
    Separates to: \(\frac{1}{y} dy = -\frac{1}{x} dx\)
  2. 2. Integrate: Integrate both sides of the equation.
    \(\int \frac{1}{y} dy = \int -\frac{1}{x} dx\)
    \(\ln|y| = -\ln|x| + C\)
  3. 3. Solve for y (Optional): You can often simplify this "general solution" using log rules or exponentiating.

Concept 2: Solving First-Order Linear Equations

This is for equations in a specific form: \(\frac{dy}{dx} + P(x)y = Q(x)\). (This applies to problems like Q19, which appears to be \(\frac{dy}{dx} - y = x\) or similar).

Steps to Solve (using \(\frac{dy}{dx} - y = x\) as example):

  1. 1. Identify P(x) and Q(x):
    The equation is \(\frac{dy}{dx} + (-1)y = x\).
    So, \(P(x) = -1\) and \(Q(x) = x\).
  2. 2. Find the Integrating Factor (I.F.): This is a special "magic" function that simplifies the problem.
    \[\text{I.F.} = e^{\int P(x) dx}\]
    \[\text{I.F.} = e^{\int -1 dx} = e^{-x}\]
  3. 3. Multiply: Multiply the *entire* original equation by the I.F.
    \[e^{-x}(\frac{dy}{dx} - y) = e^{-x}(x)\]
  4. 4. Simplify: The left side *always* magically becomes the derivative of (\(y \cdot\) I.F.).
    \[\frac{d}{dx}(y \cdot e^{-x}) = xe^{-x}\]
  5. 5. Integrate Both Sides:
    \[\int \frac{d}{dx}(y \cdot e^{-x}) dx = \int xe^{-x} dx\]
    \[y \cdot e^{-x} = \int xe^{-x} dx\]
    (You'd use Integration by Parts on the right side).
  6. 6. Solve for y: After integrating, multiply by \(e^{x}\) to get \(y\) by itself.

Concept 3: Finding Particular Solutions (IVP) & Modeling

This is about finding a *specific* solution (not just a general one with \(+ C\)) and applying these skills to word problems. (e.g., Q19, 20, 21, 22, 23).

Part A: Initial Value Problems (Q19, 20, 21)

This is just one extra step after you find the general solution.

  1. 1. Find the General Solution: Solve the DE as usual. Your answer will have a \(+ C\) in it (e.g., \(y = x + 1 + Ce^x\)).
  2. 2. Use the Initial Value: The problem gives you a point (e.g., "if \(y=1\) when \(x=0\)"). Plug these values (\(x=0, y=1\)) into your general solution.
    \[1 = (0) + 1 + Ce^{(0)}\]
  3. 3. Solve for C: \(1 = 1 + C(1) \implies C = 0\).
  4. 4. Write the Particular Solution: Rewrite your general solution with the specific value of \(C\) you found.
    Final Answer: \(y = x + 1\).

Part B: Modeling (Q22, 23)

This is about translating words into a DE, which you then solve.

  • Q22 (Growth): "rate proportional to the number present"
    Equation: \(\frac{dP}{dt} = kP\) (where \(P\) is population, \(t\) is time)
  • Q23 (Physics): "Neglecting air resistance" means acceleration is constant gravity (\(a = -g\)).
    Equation 1: \(\frac{dv}{dt} = -g\) (Solve this for \(v(t)\))
    Equation 2: \(\frac{ds}{dt} = v(t)\) (Plug in \(v(t)\) and solve this for \(s(t)\), the distance)
