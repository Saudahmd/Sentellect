1. Graphing Linear Inequalities

Visualizing Inequality Regions

Graphing linear inequalities helps us visualize the solution set as a region on the coordinate plane. The boundary line divides the plane, and shading shows which side satisfies the inequality.

Steps for Graphing:
1. Graph the boundary line (solid for ‚â§ or ‚â•, dashed for < or >)
2. Test a point (usually (0,0)) to determine which side to shade
3. Shade the region that satisfies the inequality

2. Systems of Linear Inequalities

Finding Feasible Regions

A system of linear inequalities defines a feasible region - the area where all inequalities are satisfied simultaneously. The corner points of this region are important for optimization problems.

Steps for Solving Systems:
1. Graph each inequality on the same coordinate plane
2. Identify the overlapping shaded region
3. Find the corner points of the feasible region
üîç The feasible region is where all conditions overlap!

3. Linear Optimization

Maximizing and Minimizing Linear Functions

Linear programming involves finding the maximum or minimum value of a linear objective function subject to linear constraints. The optimal solution always occurs at a corner point of the feasible region.

Steps for Optimization:
1. Graph the constraints to find the feasible region
2. Identify all corner points of the feasible region
3. Evaluate the objective function at each corner point
4. Select the point that gives the optimal value
üèÜ The best solution is always at a corner point!
Optimal Point
Example: Maximize f(x,y) = 2x + 5y subject to 2y - x ‚â§ 8, x - y ‚â§ 4, x ‚â• 0, y ‚â• 0
Step 1: Graph the constraints to find the feasible region
Step 2: Corner points: (0,0), (0,4), (4,0), (16/3, 20/3)
Step 3: Evaluate f(x,y) at each point: f(0,0)=0, f(0,4)=20, f(4,0)=8, f(16/3,20/3)=44
Step 4: Maximum value is 44 at (16/3, 20/3)
Example: Minimize z = 2x + y subject to x + y ‚â• 3, 7x + 5y ‚â§ 35, x ‚â• 0, y ‚â• 0
Step 1: Graph the constraints to find the feasible region
Step 2: Corner points: (0,3), (5,0), (0,7), (35/7,0)
