Discuss Continuity

(i) \[f(x) = \begin{cases} 2x + 5 & \text{if } x \leq 2 \\ 4x + 1 & \text{if } x = 2 \end{cases}, \, c = 2\]

(ii) Find m and n for continuity at x = 3:

Answer: m = 1, n = 3
(iii) Find m for continuity at x = 3:

(iv) Find k for continuity at x = 2:
\[f(x) = \begin{cases} \sqrt{2x + 5} - \sqrt{x + 7}, & x \neq 2 \\ k & \text{if } x = 2 \end{cases}\]
Limit as x â†’ 2: \(\lim_{x \to 2} \frac{\sqrt{2x + 5} - \sqrt{x + 7}}{x - 2} \cdot \frac{\sqrt{2x + 5} + \sqrt{x + 7}}{\sqrt{2x + 5} + \sqrt{x + 7}}\)

= \(\lim_{x \to 2} \frac{1}{\sqrt{2x + 5} + \sqrt{x + 7}} = \frac{1}{\sqrt{9} + \sqrt{9}} = \frac{1}{6}\)
For continuity: k = \(\frac{1}{6}\)
Answer: k = \(\frac{1}{6}\)
