Graphing Linear Inequalities

Graphing Single Inequalities

Learn how to graph linear inequalities on the coordinate plane.

üöß
Fence Boundaries
Visualization: Think of the boundary line as a fence. The inequality tells you which side of the fence is allowed.
üß™
Test Point Method
Easy method: Pick a test point (usually (0,0) if it's not on the line) and see if it satisfies the inequality.
üé®
Shading Regions
Remember: Shade the region that contains all points that satisfy the inequality. This is the solution set.
Graphing Steps:
1. Graph the boundary line (solid for ‚â§ or ‚â•, dashed for < or >)
2. Choose a test point not on the line (usually (0,0) if possible)
3. If the test point satisfies the inequality, shade that side; otherwise, shade the opposite side
Example: Graph 3x + 7y < 21
Step 1: Graph the boundary line 3x + 7y = 21
Step 2: Find x-intercept: set y=0 ‚Üí 3x=21 ‚Üí x=7 ‚Üí point (7,0)
Step 3: Find y-intercept: set x=0 ‚Üí 7y=21 ‚Üí y=3 ‚Üí point (0,3)
Step 4: Draw a dashed line (since < does not include equality)
Step 5: Test point (0,0): 3(0)+7(0)=0 < 21 ‚Üí True, so shade region containing (0,0)
Example: Graph x - y < 3
Step 1: Graph the boundary line x - y = 3
Step 2: Find x-intercept: set y=0 ‚Üí x=3 ‚Üí point (3,0)
Step 3: Find y-intercept: set x=0 ‚Üí -y=3 ‚Üí y=-3 ‚Üí point (0,-3)
Step 4: Draw a dashed line (since < does not include equality)
Step 5: Test point (0,0): 0-0=0 < 3 ‚Üí True, so shade region containing (0,0)
Example: Graph 3x + 2y > 6
Step 1: Graph the boundary line 3x + 2y = 6
Step 2: Find x-intercept: set y=0 ‚Üí 3x=6 ‚Üí x=2 ‚Üí point (2,0)
Step 3: Find y-intercept: set x=0 ‚Üí 2y=6 ‚Üí y=3 ‚Üí point (0,3)
Step 4: Draw a dashed line (since > does not include equality)
Step 5: Test point (0,0): 3(0)+2(0)=0 > 6 ‚Üí False, so shade region NOT containing (0,0)

Systems of Linear Inequalities

Graphing Systems and Finding Feasible Regions

Find the solution set (feasible region) for systems of linear inequalities.

üß©
Puzzle Pieces
Visualization: Each inequality is like a puzzle piece. The feasible region is where all pieces overlap.
üéØ
Intersection
Key concept: The solution to a system is the intersection (overlap) of all individual solution sets.
üìç
Corner Points
Important: The corner points of the feasible region are found by solving systems of equations from the boundary lines.
Feasible Region: The set of all points that satisfy all inequalities in a system simultaneously.
Corner Points: Points where boundary lines intersect. These are important in optimization problems.
