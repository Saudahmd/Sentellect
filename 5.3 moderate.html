Introduction to Linear Programming

What is Linear Programming?

Linear programming is a mathematical method to achieve the best outcome (maximum profit or lowest cost) in a mathematical model whose requirements are represented by linear relationships.

üè≠
Factory Production
Real-life example: A factory wants to maximize profit by deciding how many units of each product to make, given limited resources like labor and materials.
ü•ó
Diet Planning
Real-life example: Create the cheapest diet that meets all nutritional requirements by selecting the right combination of foods.
üöö
Transportation
Real-life example: Find the most cost-effective way to transport goods from multiple factories to multiple stores.
Linear Programming Problem Components:
1. Objective Function: The function to maximize or minimize (e.g., profit, cost)
2. Constraints: Limitations or requirements (e.g., resource limits, minimum requirements)
3. Decision Variables: The quantities we can control (e.g., x, y)
4. Feasible Region: The set of all points satisfying all constraints
5. Optimal Solution: The point in the feasible region that optimizes the objective function
Key Theorem: Corner Point Method

If a linear programming problem has an optimal solution, it must occur at a corner point of the feasible region.

Step 1: Graph all constraints to find the feasible region
Step 2: Identify all corner points of the feasible region
Step 3: Evaluate the objective function at each corner point
Step 4: Select the corner point that gives the optimal value
Example: Simple Maximization

Maximize P = 3x + 2y subject to:
x + y ‚â§ 4
x ‚â• 0, y ‚â• 0

Step 1: Feasible region is a triangle with corners at (0,0), (4,0), (0,4)
Step 2: Evaluate P at each corner:
At (0,0): P = 3(0) + 2(0) = 0
At (4,0): P = 3(4) + 2(0) = 12
At (0,4): P = 3(0) + 2(4) = 8
Step 3: Maximum value is 12 at point (4,0)
Example: Simple Minimization

Minimize C = x + 3y subject to:
2x + y ‚â• 6
x ‚â• 0, y ‚â• 0

Step 1: Feasible region is unbounded with corners at (3,0), (0,6)
Step 2: Evaluate C at each corner:
At (3,0): C = 3 + 3(0) = 3
At (0,6): C = 0 + 3(6) = 18
Step 3: Minimum value is 3 at point (3,0)

Solving Maximization Problems

üí∞
Profit Maximization
Business application: Companies use linear programming to maximize profits while considering production limits, resource constraints, and market demands.
‚ö°
Efficiency
Key insight: The optimal solution is always at a corner point, which makes the problem computationally manageable.
üìä
Graphical Method
For 2 variables: We can solve maximization problems graphically by plotting constraints and finding the corner point with the highest objective function value.
Maximization Problem Steps:
1. Identify the objective function to maximize
2. Write all constraints as linear inequalities
3. Graph the feasible region
4. Find all corner points
5. Evaluate the objective function at each corner point
6. Select the point with the maximum value
Problem 1: Maximize f(x,y) = 2x + 5y

Subject to:
2y - x ‚â§ 8
x - y ‚â§ 4
x ‚â• 0, y ‚â• 0

Step 1: Graph the constraints to find the feasible region
Step 2: Find intersection points:
Line 1: 2y - x = 8 ‚Üí x = 2y - 8
Line 2: x - y = 4 ‚Üí x = y + 4
Step 3: Solve system: 2y - 8 = y + 4 ‚Üí y = 12, x = 16
Step 4: Corner points: (0,0), (0,4), (16,12), (4,0)
Step 5: Evaluate f(x,y) at each point:
(0,0): 0
(0,4): 20
(16,12): 92
(4,0): 8
Step 6: Maximum value is 92 at (16,12)
Problem 2: Maximize f(x,y) = x + 3y

Subject to:
2x + 5y ‚â§ 30
5x + 4y ‚â§ 20
x ‚â• 0, y ‚â• 0

Step 1: Graph the constraints to find the feasible region
Step 2: Find intersection of 2x+5y=30 and 5x+4y=20
Step 3: Solve system: Multiply first equation by 5 and second by 2:
10x + 25y = 150
10x + 8y = 40
Subtract: 17y = 110 ‚Üí y ‚âà 6.47, x ‚âà -1.18
Step 4: But x cannot be negative, so check boundary points:
(0,0): f=0
(0,6): f=18
(4,0): f=4
Step 5: Maximum value is 18 at (0,6)
Problem 3: Maximize z = 2x + 3y

Subject to:
3x + 4y ‚â§ 12
2x + y ‚â§ 4
4x - y ‚â§ 4
x ‚â• 0, y ‚â• 0

Step 1: Graph all constraints to find the feasible region
Step 2: Find all intersection points:
3x+4y=12 and 2x+y=4 ‚Üí (0.8, 2.4)
3x+4y=12 and 4x-y=4 ‚Üí (1.45, 1.91)
2x+y=4 and 4x-y=4 ‚Üí (1.33, 1.33)
Step 3: Corner points: (0,0), (0,3), (1,0), (0.8,2.4), (1.33,1.33)

(0,0): 0
(0,3): 9
(1,0): 2
(0.8,2.4): 8.8

Solving Minimization Problems

Learn how to find minimum values of objective functions subject to constraints.

Cost Minimization
Business application: Companies use linear programming to minimize costs while meeting production requirements, quality standards, and delivery deadlines.

Unbounded Solutions
Important note: Minimization problems with unbounded feasible regions may not have optimal solutions if the objective function can decrease without bound.
Minimization Problem Steps:
1. Identify the objective function to minimize
2. Write all constraints as linear inequalities
3. Graph the feasible region
4. Find all corner points
5. Evaluate the objective function at each corner point
6. Select the point with the minimum value
Problem 1: Minimize z = 2x + y

Subject to:
x + y ‚â• 3
7x + 5y ‚â§ 35
x ‚â• 0, y ‚â• 0

Step 1: Graph the constraints to find the feasible region
Step 2: Find intersection of x+y=3 and 7x+5y=35
Step 3: Solve system: From x+y=3, y=3-x
Substitute: 7x+5(3-x)=35 ‚Üí 7x+15-5x=35 ‚Üí 2x=20 ‚Üí x=10, y=-7
Step 4: But y cannot be negative, so check boundary points:
Intersection of x+y=3 and y=0: (3,0)
Intersection of 7x+5y=35 and x=0: (0,7)
Intersection of 7x+5y=35 and y=0: (5,0)
Step 5: Evaluate z at each point:
(3,0): z=6
(0,7): z=7
(5,0): z=10
Step 6: Minimum value is 6 at (3,0)
Problem 2: Minimize z = 3x + y

Subject to:
3x + 5y ‚â• 15
x + 6y ‚â• 9
x ‚â• 0, y ‚â• 0

Step 1: Graph the constraints to find the feasible region
Step 2: Find intersection of 3x+5y=15 and x+6y=9
Step 3: Solve system: From x+6y=9, x=9-6y
Substitute: 3(9-6y)+5y=15 ‚Üí 27-18y+5y=15 ‚Üí -13y=-12 ‚Üí y=12/13‚âà0.92, x=9-6(12/13)=45/13‚âà3.46
Step 4: Corner points: (5,0), (0,2.5), (3.46,0.92)
Step 5: Evaluate z at each point:
(5,0): z=15
(0,2.5): z=2.5
(3.46,0.92): z=11.3
Step 6: Minimum value is 2.5 at (0,2.5)

Application 1: Diet Problem
Each unit of food X costs Rs. 25 and contains 2 units of protein and 4 units of iron.
Each unit of food Y costs Rs. 30 and contains 3 units of protein and 2 units of iron.
Each animal must receive at least 12 units of protein and 16 units of iron each day.
How many units of each food should be fed to each animal at the smallest possible cost?
Step 1: Define variables:
Let x = units of food X
Let y = units of food Y
Step 2: Objective function (minimize cost):
C = 25x + 30y
Step 3: Constraints:
Protein: 2x + 3y ‚â• 12
Iron: 4x + 2y ‚â• 16
Non-negativity: x ‚â• 0, y ‚â• 0
Step 4: Find corner points:
Intersection of 2x+3y=12 and 4x+2y=16
Solve: Multiply first equation by 2: 4x+6y=24
Subtract second equation: 4y=8 ‚Üí y=2, then x=3
Step 5: Other corner points: (0,6), (4,0), (8,0)
Step 6: Evaluate C at each point:
(3,2): C=25(3)+30(2)=135
(0,6): C=25(0)+30(6)=180
(4,0): C=25(4)+30(0)=100
(8,0): C=25(8)+30(0)=200
Step 7: Minimum cost is Rs. 100 with 4 units of food X and 0 units of food Y
Application 2: Production Planning
A machine can produce product A using 2 units of chemical and 1 unit of compound,
or product B using 1 unit of chemical and 2 units of compound.
Only 800 units of chemical and 1000 units of compound are available.
The profits per unit of A and B are Rs. 30 and Rs. 20 respectively.
Maximize the profit function.
Step 1: Define variables:
Let x = units of product A
Let y = units of product B
Step 2: Objective function (maximize profit):
P = 30x + 20y
Step 3: Constraints:
Chemical: 2x + y ‚â§ 800
Compound: x + 2y ‚â§ 1000
Non-negativity: x ‚â• 0, y ‚â• 0
Step 4: Find corner points:
Intersection of 2x+y=800 and x+2y=1000
Solve: Multiply second equation by 2: 2x+4y=2000
Subtract first equation: 3y=1200 ‚Üí y=400, then x=200
Step 5: Other corner points: (0,0), (400,0), (0,500)
Step 6: Evaluate P at each point:
(0,0): P=0
(400,0): P=30(400)+20(0)=12000
(0,500): P=30(0)+20(500)=10000
(200,400): P=30(200)+20(400)=6000+8000=14000
Step 7: Maximum profit is Rs. 14,000 with 200 units of A and 400 units of B
