Translation of Axes

Moving the Coordinate System

Shift the origin to a new location without changing axes direction.

Understanding Translation of Axes

What is Translation of Axes?

Translation of axes means moving the origin to a new point O'(h,k) while keeping the axes parallel to their original directions. This is useful when we want to simplify equations or analyze geometric figures from a different perspective.

How Does Translation Work?

When we translate the coordinate system, the relationship between old coordinates (x,y) and new coordinates (X,Y) is given by:

\[ X = x - h \]

\[ Y = y - k \]

Where (h,k) are the coordinates of the new origin in the old coordinate system.

Visual Representation

Imagine shifting the entire coordinate grid so that the origin moves to a new location. All points keep their relative positions, but their coordinates change according to the new origin.

This transformation is particularly useful for simplifying equations of curves and analyzing symmetry.

Find new coordinates of P(3,2) when origin is shifted to O'(1,3).

Rotation of Axes

Turning the Coordinate System

Rotate the axes while keeping the origin fixed.

Understanding Rotation of Axes

What is Rotation of Axes?

Rotation of axes means turning the coordinate system about the origin through an angle Œ∏ while keeping the origin fixed. This is useful for simplifying equations, especially those containing xy terms.

How Does Rotation Work?

When we rotate the coordinate system through an angle Œ∏, the relationship between old coordinates (x,y) and new coordinates (X,Y) is given by:

\[ X = x \cos\theta + y \sin\theta \]

\[ Y = y \cos\theta - x \sin\theta \]

These formulas come from considering the projections of the point onto the rotated axes.

Visual Representation

Imagine rotating the entire coordinate grid about the origin. The axes change direction, but all points maintain their positions in space. Their coordinates change according to the new orientation of the axes.

This transformation is particularly useful for eliminating the xy term in the general second-degree equation.

Combined Transformations

Translation + Rotation

Apply both translation and rotation to coordinate systems.

Understanding Combined Transformations

What are Combined Transformations?

Combined transformations involve applying both translation and rotation to the coordinate system. This is useful when we want to both change the origin and reorient the axes.

How Do Combined Transformations Work?

When we apply both translation and rotation, we typically:

1. First translate the origin to O'(h,k)

2. Then rotate the axes through angle Œ∏

The combined transformation formulas are:

\[ X = (x - h) \cos\theta + (y - k) \sin\theta \]

\[ Y = (y - k) \cos\theta - (x - h) \sin\theta \]

Visual Representation

Imagine first shifting the coordinate grid to a new origin, then rotating the axes about this new origin. This gives us maximum flexibility in positioning and orienting our coordinate system.

This transformation is particularly useful for analyzing curves and surfaces from different perspectives.

Key Differences

Translation vs Rotation

Understand when to use each transformation and how they differ.

Understanding the Differences

Fundamental Differences

Translation and rotation are two distinct types of coordinate transformations with different purposes and effects:

When to Use Each Transformation

Translation: Use when you want to change the reference point (origin) but keep the same orientation. This is useful for simplifying equations by removing constant terms.

Rotation: Use when you want to change the orientation of the axes but keep the same origin. This is useful for eliminating xy terms from equations.

Combined: Use when you need both a new reference point and a new orientation. This gives maximum flexibility in positioning your coordinate system.

Visual Comparison

In translation, the entire coordinate grid shifts to a new position. In rotation, the grid pivots about the origin. Combined transformations involve both shifting and pivoting.

Understanding these differences helps you choose the right transformation for your specific problem.
